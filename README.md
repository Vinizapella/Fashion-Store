# 🛍️ Fashion Store

> Uma loja de moda moderna, elegante e totalmente responsiva.

---

## 🗂️ Sobre

O **Fashion Store** é uma aplicação web de e-commerce voltada para o segmento de moda, apresentando produtos, coleções e uma experiência de navegação fluida e visualmente atraente — do vitrine ao carrinho.

---

## ✨ Funcionalidades

- 👗 Vitrine de produtos com categorias
- 🔍 Busca e filtragem de itens
- 🛒 Carrinho de compras interativo
- 📱 Layout responsivo para mobile e desktop
- 🖼️ Galeria de imagens dos produtos
- 💳 Página de checkout

---

## 🛠️ Tecnologias utilizadas

| Tecnologia | Função |
|---|---|
| 🌐 HTML5 | Estrutura das páginas |
| 🎨 CSS3 | Estilização e responsividade |
| ⚡ JavaScript | Interatividade e carrinho |

---

## 🚀 Como executar

```bash
# Clone o repositório
git clone https://github.com/Vinizapella/Fashion-Store.git

# Acesse a pasta
cd Fashion-Store
```

> Abra o arquivo `index.html` diretamente no navegador — sem necessidade de servidor! 🌐
